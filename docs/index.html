Syncopated ISO

The "Syncopated ISO" project is a custom, Arch Linux-based, bootable Live/Rescue ISO image. It provides a pre-configured environment for system administration, recovery tasks, or as a platform for installing a customized Arch Linux system using its integrated installer script. It's a spiritual successor to ArchLabs Linux, maintaining a minimalist aesthetic and user control.

Features

>
  • ArchLabs Heritage: A spiritual successor to the defunct ArchLabs project.
  • Arch Linux Base: Built upon the latest Arch Linux packages.
  • Archiso Framework: Uses the official Archiso tools for building.
  • Broad Boot Support: Boots on both legacy BIOS (Syslinux) and modern UEFI (GRUB) systems.
  • Custom Theming: Includes themes like `oomox-soundbot`.
  • Pre-configured Tools: Comes with Rofi, Ranger, Nitrogen, `sxhkd`, etc.
  • Installer: Provides a custom installation script (`installer.sh`).
  • Security Aware: Sets specific file permissions.

Practical Use-Cases

>
  • System Administration: Boot into a live environment with essential tools for managing an existing Arch Linux system.
  • System Recovery: Utilize the pre-configured environment to troubleshoot boot issues, repair file systems, or restore backups.
  • Customized Arch Linux Installation: Use the integrated `installer.sh` script to perform a fresh installation of Arch Linux with a pre-defined set of configurations and packages.
  • Portable Environment: Run a consistent Arch Linux environment directly from USB on various machines without affecting their local installations.
  • Development & Testing: Provide a clean, reproducible Arch Linux base for testing scripts, playbooks, or new software configurations before deploying to production systems.
ℹ️ Important Note: The integrated `installer.sh` is a powerful script that will modify your hard drive. It prompts for confirmation multiple times before making significant changes, but it's crucial to exercise caution. Please ensure you have backed up all important data and reviewed the script's behavior before proceeding.

Ansible Workstation Archlinux

This Ansible playbook automates the configuration of a development-focused workstation, primarily targeting Arch Linux and its derivatives. It leverages Ansible roles for modularity and maintainability, setting up everything from base system utilities to a complete development environment. It supports both standard `ansible-playbook` execution and `ansible-pull` for agent-based configuration management.

Key Features

>
  • Base System: Package manager config, repositories, user creation, sudo.
  • Shell Environment: Zsh, Oh My Zsh, Kitty, Ranger.
  • Containerization & Virtualization: Docker (with NVIDIA), Libvirt, Vagrant.
  • Desktop Environment: X11 (i3, Rofi, Picom), Wayland (Sway, XWayland), GRUB theming.
  • Development Tools: Ruby, Go, Fabric (AI scripting), custom tools.
  • Networking & Services: Unified network config, automatic Wi-Fi, Firewalld, OpenSSH, NTP.
  • Professional Audio: Configurable audio server (JACK, PulseAudio, PipeWire), low-latency tuning.
  • Dynamic Inventory: Enhanced inventory management with group/host variables.

Modular Roles Overview

>

The playbook is organized into modular roles for various functionalities, ensuring maintainability and reusability. Examples include:

  • `audio`: Professional Linux audio setup.
  • `base`: Core system setup, package management.
  • `desktop`: General desktop applications, fonts, themes.
  • `docker`: Docker Engine & Docker Compose.
  • `i3`: i3 Window Manager and related tools.
  • `network`: Comprehensive network management.
  • `shell`: Zsh, Oh My Zsh, Kitty, Ranger.
  • `sway`: Tiling Window Manager for Wayland.
  • And many more, each handling a specific aspect of the workstation configuration.

The Power of Integration: ISO & Ansible

The Syncopated ISO and Syncopated Ansible projects are designed to work hand-in-hand, providing a powerful and streamlined approach to Arch Linux workstation deployment and management.

Here's how they integrate:

  1. Initial System Installation with Syncopated ISO:

    The Syncopated ISO provides a bootable live environment with an integrated `installer.sh` script. This script allows you to quickly install a foundational Arch Linux system onto your hardware. It handles partitioning, basic system setup, and ensures a minimal yet functional Arch installation.

  2. Automated Workstation Configuration with Syncopated Ansible:

    Once the basic Arch Linux system is installed using the ISO, the Syncopated Ansible playbook takes over. You can run this playbook on the newly installed system (or any existing Arch Linux machine) to transform it into a fully configured, development-focused workstation.

    The Ansible playbook automates the installation of all necessary software, configuration of desktop environments (X11 with i3 or Wayland with Sway), shell environments, development tools, containerization (Docker, Libvirt), networking, professional audio settings, and much more, based on your defined variables and roles.

  3. Continuous Management:

    The Ansible playbook can also be used for ongoing configuration management, allowing you to easily update, modify, or re-apply your desired workstation setup across multiple machines, ensuring consistency and reducing manual effort.

This integration provides a robust solution: the ISO gets you a working Arch base quickly, and Ansible then builds out your complete, customized development environment with precision and repeatability.
